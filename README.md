# Funkollection 🎉

Funkollection is a web application designed to help Funko Pop collectors manage their collection. Users can add, view, and organize their Funko Pop figures in a personalized and user-friendly interface.

## Features ✨

- **User Authentication**: Secure login and registration using ASP.NET Core Identity.
- **Funko Pop Management**: Add, view, and delete Funko Pops in your collection.
- **Sticker Tracking**: Associate Funko Pops with various stickers.
- **Search and Filter**: Easily find Funko Pops using search functionality.
- **User Collection**: Track which users own which Funko Pops.
- **Responsive Design**: Fully responsive layout that works on both desktop and mobile devices.

## Technologies Used 🛠️

- **Frontend**: Razor Pages with Bootstrap for styling.
- **Backend**: ASP.NET Core with Razor Pages.
- **Database**: Entity Framework Core with SQL Server.
- **Authentication**: ASP.NET Core Identity.
- **Styling**: Bootstrap for a clean and responsive UI.

## Usage 🏗️

- **Sign up or log in** to start managing your Funko Pop collection.  
- **Add new Funko Pops** with details like name, series, title, and stickers.  
- **View and search** for Funko Pops in your collection.  
- **Filter by stickers** to easily find specific Funko Pops.  
- **Manage your profile** to keep track of your collection stats.  

## Known Issues & Bugs 🐛
